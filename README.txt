
Plan SESCAM Marisa - Web version (static)
-----------------------------------------

Estructura:
- index.html
- styles.css
- app.js
- sounds/estudio.wav, sounds/descanso.wav

Modo de uso local:
1. Descomprime esta carpeta en tu equipo.
2. Abre index.html en tu navegador (Chrome/Edge/Firefox).
3. La app usa localStorage para guardar el progreso.

Despliegue en GitHub Pages (recomendado):
1. Crea un repositorio en GitHub llamado 'plan-sescam-marisa'.
2. Subir todos los archivos (index.html, styles.css, app.js, sounds/).
3. Ve a Settings -> Pages -> Branch: main (o master) -> / (root) y guarda.
4. Tu página estará disponible en: https://marielha2021.github.io/plan-sescam-marisa/

Notas:
- Los tests y simulacros se abren en ChatGPT (en este chat). La app muestra botones y guarda resultados manualmente.
- Si quieres integración automática de tests, puedo añadir un backend más adelante.
